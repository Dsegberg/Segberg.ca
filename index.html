Smart Seat

Smart Seat is a classroom web app for fast tap-in attendance, instant seat assignment, and live in/out tracking — designed to be readable from anywhere in the room.

Launch Smart Seat Who it’s for
RFID / NFC tap-in Big-screen friendly Google Sheets backend Apps Script web app

Who it’s for

Built to support real routines — daily attendance, labs, exam seating, and sub days.

Teachers Run attendance and seating fast without interrupting instruction.
Exam Supervision Quick seating changes, clear layouts, and less confusion.
Substitutes Simple “launch-and-use” workflow with clear visibility.

How it works

No extra platforms. Uses simple, school-friendly tools.

RFID / NFC Students tap a card or tag for quick check-in.
Google Sheets School-owned data backend for easy updates and exports.
Google Apps Script Web app delivery with controlled access and PIN protection (inside each tool).

Each tool opens as a web app, while your data stays organized behind the scenes in Google Sheets.
